# RSA Криптосистема с открытым ключом — Вариант 1

## Как запустить 

Ввести в bash ./lab3.

## Функции интерфейса

- Выбор режима: **Зашифровать** / **Расшифровать**
- Загрузка файла любого формата (drag & drop или клик)
- Отображение первых 40 блоков: ключевого потока, исходного файла и результата
- Скачивание результирующего файла (поток байт)
