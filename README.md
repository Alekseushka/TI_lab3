# LFSR Поточное Шифрование — Вариант 1

Веб-приложение на Go для потокового шифрования файлов с помощью LFSR.

## Параметры варианта

| Параметр              | Значение                  |
|-----------------------|---------------------------|
| Степень многочлена    | 23                        |
| Примитивный многочлен | x²³ + x⁵ + 1             |
| Отводы обратной связи | бит 23 ⊕ бит 5            |
| Период                | 2²³ − 1 = 8 388 607 бит   |


## Алгоритм работы LFSR

Регистр состоит из 23 бит: `reg[0]` — старший (x²³), `reg[22]` — младший (x¹).

На каждом такте:
1. Выходной бит = `reg[22]` (LSB)
2. Бит обратной связи = `reg[0] XOR reg[18]` (отводы x²³ и x⁵)
3. Регистр сдвигается вправо, в `reg[0]` записывается бит обратной связи

Шифрование: `C = M XOR KeyStream`  
Дешифрование: `M = C XOR KeyStream` (операция симметрична при одинаковом начальном состоянии)

## Функции интерфейса

- Выбор режима: **Зашифровать** / **Расшифровать**
- Ввод начального состояния (23 бита): поле игнорирует любые символы кроме `0` и `1`
- Загрузка файла любого формата (drag & drop или клик)
- Отображение первых 256 бит: ключевого потока, исходного файла и результата
- Скачивание результирующего файла (поток байт)
